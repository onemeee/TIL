### 2차원 배열 list
#### 잘못된 방법
```python
arr = [[0] * cols] * rows
```
다음과 같이 배열을 주게되면, 얕은 복사가 이루어진다.  
따라서 arr[0][0] = 1을 하였을 때 arr[1][0] ~ arr[-1][[0]까지가 전부 변경된다!
#### 정확한 방법
```python
arr = [[0 for j in range(cols)] for i in range(rows)]
```
다음과 같은 방법으로 배열을 주면 문제 해결!
